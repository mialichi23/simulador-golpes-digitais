Simulador de Golpes Digitais

Descrição

Este projeto consiste no desenvolvimento de uma aplicação em Python voltada à conscientização sobre golpes digitais. A aplicação simula diferentes tipos de mensagens comumente utilizadas em fraudes online, permitindo que o usuário identifique se o conteúdo apresentado é legítimo ou potencialmente malicioso.

O sistema fornece feedback imediato com explicações, contribuindo para a educação digital e o desenvolvimento do senso crítico do usuário frente a tentativas de engenharia social.

⸻

Objetivo

Desenvolver uma ferramenta interativa que auxilie na identificação de golpes digitais, com foco em educação e prevenção, especialmente para usuários com menor familiaridade com tecnologia.

⸻

Funcionamento

A aplicação apresenta uma sequência de mensagens simuladas (inspiradas em contextos reais, como bancos, serviços de entrega e comunicações pessoais). Para cada mensagem, o usuário deve classificá-la como:

	•	“golpe”
	•	“seguro”

Após cada resposta, o sistema informa se a escolha foi correta e apresenta uma explicação contextualizada.

⸻

Funcionalidades

	•	Simulação de cenários reais de fraude digital
	•	Classificação interativa de mensagens
	•	Feedback explicativo para reforço do aprendizado
	•	Cálculo de pontuação ao final da execução
	•	Reexecução do fluxo de interação

⸻

Tecnologias utilizadas

	•	Python 3.10 ou superior

⸻

Execução

Para executar o projeto, utilize o seguinte comando no terminal:

python main.py

Estrutura do projeto

O código foi organizado de forma modular, com separação de responsabilidades por meio de funções, incluindo:

	•	exibição da interface textual
	•	coleta e validação de entrada do usuário
	•	execução da lógica de avaliação
	•	apresentação de resultados

Essa organização segue princípios básicos de legibilidade e manutenção de código.

⸻

Possíveis extensões

Como evolução do projeto, destacam-se:

	•	Implementação de interface gráfica ou web
	•	Inclusão de múltiplos idiomas
	•	Classificação por tipo de golpe (phishing, engenharia social, etc.)
	•	Armazenamento de desempenho do usuário
	•	Adaptação do conteúdo para públicos específicos (ex: idosos)
